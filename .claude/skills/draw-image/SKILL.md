# draw-image: OpenAI gpt-image-1 图像生成

## 适用范围（MUST READ）

**应该用本 skill 生成的图：**

| 类型 | 示例 |
|------|------|
| 算法 / 代码流程图 | 建模流程、数据处理管线、求解步骤 |
| 系统架构示意图 | Agent 调用链、模块关系图 |
| 概念性插图 | 物理场景示意、问题背景图 |
| 方法对比示意图 | 两种模型的直觉对比 |

**不应该用本 skill 的图（必须跑代码生成）：**

- 数据分布图 / 散点图 / 折线图（源自真实数据）
- 模型求解结果可视化（优化轨迹、拟合曲线等）
- 灵敏度分析图表
- 任何需要体现代码实际运行数值的图

## 调用方法

### 基础调用

```bash
python scripts/draw_image.py \
  --prompt  "Clean white-background flowchart showing: ..." \
  --output  "CUMCM_Workspace/latex/images/fig_flowchart.png"
```

### 参数说明

| 参数 | 默认值 | 选项 | 说明 |
|------|--------|------|------|
| `--prompt` | — | — | 图像描述（见下方 Prompt 指南） |
| `--output` | — | — | 保存路径，建议 `.png` |
| `--size` | `1024x1024` | `1024x1024` `1536x1024` `1024x1536` `auto` | 横图用 1536x1024，竖图用 1024x1536 |
| `--quality` | `medium` | `low` `medium` `high` `auto` | 论文插图建议 `high` |
| `--model` | `gpt-image-1` | — | OpenAI 图像模型 |

## Prompt 工程指南

### 流程图（Flowchart）

```
Prompt 结构：
"Clean technical flowchart on white background. [语言说明]. 
Steps: (1) [步骤一] → (2) [步骤二] → ... 
Decision nodes: [条件判断]. 
Style: professional diagram, minimal, clear labels, sans-serif font, 
blue/gray color scheme, arrows clearly show flow direction."
```

示例（算法流程图）：

```bash
python scripts/draw_image.py \
  --prompt "Clean technical flowchart on white background, Chinese labels. 
Steps: (1) 读取题目 → (2) 识别模型类型 → (3) 文献调研 → (4) 编写求解代码 
→ (5) 运行验证脚本 → Decision: 全部通过? → Yes: 写入LaTeX / No: 修复代码回到(4). 
Style: professional, minimal, blue/gray, sans-serif, clear arrows." \
  --output "CUMCM_Workspace/latex/images/fig00_pipeline_flow.png" \
  --size 1024x1536 \
  --quality high
```

### 系统架构图（Architecture Diagram）

```
"System architecture diagram on white background. 
Components: [组件列表]. 
Connections: [连接关系]. 
Style: clean, labeled boxes with arrows, professional tech diagram."
```

### 概念插图（Conceptual Illustration）

```
"Simple conceptual illustration: [场景描述]. 
Technical/scientific style, clean white background, labeled elements, 
no text-heavy, suitable for academic paper figure."
```

## LaTeX 集成

生成图像后，在 LaTeX 中插入：

```latex
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.85\textwidth]{images/fig_flowchart.png}
  \caption{[图题]}
  \label{fig:flowchart}
\end{figure}
```

**注意**：`images/` 路径相对于 `latex/` 目录。若 `\graphicspath` 已配置，使用相对于该路径的文件名。

## 错误处理

| 错误信息 | 解决方案 |
|---------|---------|
| `OPENAI_API_KEY not set` | `export OPENAI_API_KEY=sk-...` |
| `openai package not installed` | `pip install openai` |
| `API returned no image data` | 检查 prompt 是否触发内容策略（避免真实人物、版权图像描述） |
| `content_policy_violation` | 简化 prompt，移除可能敏感的描述 |

## 批量生成（多张图）

```bash
# 为同一论文生成多张流程图
for cfg in \
  "fig01_model_flow|建模流程图|1024x1536" \
  "fig02_data_pipeline|数据处理管线|1536x1024" \
  "fig03_system_arch|系统架构示意|1536x1024"; do
  IFS='|' read -r name desc size <<< "$cfg"
  python scripts/draw_image.py \
    --prompt "Clean professional flowchart: ${desc}, white background, Chinese labels, blue/gray color scheme" \
    --output "CUMCM_Workspace/latex/images/${name}.png" \
    --size "$size" --quality high
done
```
