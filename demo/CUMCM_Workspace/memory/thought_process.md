### 三、有效遮蔽判定准则（核心物理机制）

**"云团中心10m范围内的烟幕浓度在起爆20s内可为目标提供有效遮蔽"**

几何解释：
- 起爆时刻 t_det 后，云团中心 C(t) = (cx, cy, cz_det - 3*(t-t_det))，对 t ∈ [t_det, t_det+20]
- 导弹 M(t) 飞向假目标
- **有效遮蔽条件**：云团球体（半径10m）位于导弹与真目标连线上
  - 即从导弹位置 M(t) 到真目标中心 TT=(0,200,5) 的线段，其与云团中心距离 ≤ 10m
  - 且云团在线段范围内（参数 s ∈ [0,1]）

注：当云团恰在导弹附近时，两种判据（导弹穿过云团 vs 云团遮挡视线）几乎等价。
优先使用"遮挡线对视线"判据，更符合导引头被遮蔽的物理含义。

### 五、关键数学公式

**导弹轨迹（匀速直线）：**
M_i(t) = M_i(0) + (t/|M_i(0)|) * (-M_i(0)) * 300
即 M_i(t) = M_i(0) * (1 - 300t/|M_i(0)|)

**FY_j 轨迹（等高度匀速直线）：**
FY_j(t) = FY_j(0) + v_j * t * dir_j(水平)，z_j 恒定

**干扰弹抛体运动（投放时刻 t_rel，投放点 P_rel，初速 vel_drone）：**
弹位置(t): x = P_rel.x + vel_drone.x*(t-t_rel)
           y = P_rel.y + vel_drone.y*(t-t_rel)
           z = P_rel.z - 0.5*g*(t-t_rel)²

**起爆点（τ_det 秒后）：**
Det = (P_rel.x + vel_drone.x*τ_det, P_rel.y + vel_drone.y*τ_det, P_rel.z - 0.5*g*τ_det²)
t_det = t_rel + τ_det

**云团中心（起爆后 Δt 秒）：**
C(t) = (Det.x, Det.y, Det.z - 3*Δt)，Δt = t - t_det ∈ [0, 20]

**遮蔽条件（云团遮挡导弹到真目标视线）：**
TT = (0, 200, 5)（真目标中心）
d = TT - M(t)
v = C(t) - M(t)
s = (v·d) / |d|²
dist = |v - s*d|
条件：dist ≤ 10 AND s ∈ [0, 1]
